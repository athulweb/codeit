1. Basic Structure of HTML


<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>

2. Headings & Paragraphs


<h1>Main Heading</h1>
<h2>Subheading</h2>
<p>This is a paragraph of text.</p>

3. Links & Images


<a href="https://example.com">Click me</a>
<img src="image.jpg" alt="My Image">

4. Lists (ul, ol, li)


<ul>
  <li>Item One</li>
  <li>Item Two</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

5. Tables


<table border="1">
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>Athul</td><td>13</td></tr>
</table>

6. Forms


<form>
  <input type="text" placeholder="Name">
  <input type="email" placeholder="Email">
  <button type="submit">Send</button>
</form>

7. Media (Video, Audio)


<video src="movie.mp4" controls></video>
<audio src="song.mp3" controls></audio>

8. Divs & Spans (Layout)


<div>I am a block</div>
<span>I am inline</span>

9. Semantic Tags


<header></header>
<nav></nav>
<main></main>
<section></section>
<article></article>
<footer></footer>

10. Meta Tags (for SEO)


<meta name="description" content="Learn HTML in 1 page">
<meta name="keywords" content="HTML, Web Development">
<meta name="author" content="Athul">

11. Inline vs Block Tags

  • Inline: span, a, img, input
  • Block: div, p, h1-h6, section
